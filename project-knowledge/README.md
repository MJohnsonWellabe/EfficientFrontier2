# project-knowledge/

`EfficientFrontier-Knowledge.md` is a single, self-contained reference (design, mechanics, validated
numbers, configuration, and the full engine source) for use as **Claude Project knowledge** — so you can
ask research questions about the model in a Project without running code.

To use: drag `EfficientFrontier-Knowledge.md` into your Claude Project's knowledge (or point the
Project's GitHub connector at this repo). Regenerate it whenever the engine/canon changes.
