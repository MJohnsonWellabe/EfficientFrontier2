# data/ — workbook-derived inputs (not embedded in code)

Extracted verbatim from `legacy/EfficientFrontier-29.html`'s embedded blob.

- `InputEV.csv` — per-cohort monthly EV variables (EarnedPrem, IncClaims, reserves, lives, …).
- `InputTS.csv` — NAIC RBC charge lines by product / year-end.
- `InputSurplus.csv` — total surplus, non-insurance portion, AVR by year.
- `params.json` — assumptions (`assum`), workbook scalars (`scalars`), TS add-ins (`ts_adj`).
